## 📅 매주 (일요일 밤 또는 월요일 새벽)

### 1️⃣ Cowork에 들어가서 Claude한테 부탁

**복사해서 쓰는 마법 주문 ✨:**
```
다음 주 영어 콘텐츠 만들어줘.
- 시작 날짜: 2026-MM-DD (월요일)
- 평일 5일치 단어 20개씩 (테마는 알아서 정해줘 - 초등 4학년 수준)
- 토요일 종합시험 (그 주 100단어)
- 일요일 복습 퀴즈
```

> 💡 테마를 직접 정하고 싶으면:
> "테마는 월요일=공룡, 화요일=우주, ..." 처럼 적으면 돼요!

Claude가 자동으로:
- ✅ JSON 파일 7개 생성 (월~금 + 토 + 일)
- ✅ data/ 폴더에 저장
- ✅ 다 만들어졌다고 알려줌

### 2️⃣ 더블클릭 한 번!

```
📁 D:\Claude\05.혜원 영어\
   └─ 🖱️ "🚀_전체_업로드.bat" 더블클릭
```

자동으로:
- ✅ JSON → HTML 변환 (월~일)
- ✅ 이벤트 매니페스트 갱신
- ✅ GitHub 자동 업로드
- ✅ 1~2분 후 사이트에 반영

### 3️⃣ 끝! 🎉

혜원이는 https://hyewon-english.pages.dev/ 에서 새 콘텐츠 자동으로 봐요.

## ❓ 응급 상황: 시트에서 직접 수정

혜원이 폰의 데이터를 강제로 바꾸려면 **"동기화상태"** 시트:

| 항목 | 의미 | 수정 효과 |
|------|------|----------|
| `points` | 보유 포인트 | 즉시 반영 |
| `lucky_received_date` | 오늘 럭키박스 | 비우면 다시 받기 가능 |
| `last_streak_reward` | 마지막 스트릭 보상 | 0으로 → 다음 학습 시 또 받음 |

수정 후 혜원이 폰 새로고침 = 끝!

## 🔧 고장났을 때
