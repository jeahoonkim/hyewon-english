# 혜원이 영단어 학습 시스템 - 설정 가이드 📚

## 🎉 훨씬 간단해졌어요!

재님이 이미 기존 "매일매일 퀴즈내기"에서 카톡 발송 Apps Script를 만들어두셨기 때문에,
**카톡 토큰을 다시 찾을 필요가 없어요!** 그 URL을 그대로 재사용합니다.

그래서 이제 재님이 할 일은 딱 **Google Sheets용 Apps Script 1개 배포** 뿐이에요.

## 🎯 재님이 할 일 (딱 3단계, 약 5분)

### 1단계: Google Apps Script 생성 (3분)

**1-1.** 재님이 방금 만든 "혜원이 영단어 기록" 스프레드시트를 열어요.
(이미 열려 있죠? URL에 `1wqob-TM5Uf...` 들어있는 그 시트)

**1-2.** 상단 메뉴에서 **확장 프로그램 → Apps Script** 클릭

**1-3.** 새 창이 열리면 기본 `function myFunction() {}` 코드를 **모두 지워요**

**1-4.** `GoogleAppsScript_코드.gs` 파일을 메모장으로 열고 **전체 복사 (Ctrl+A → Ctrl+C)**

**1-5.** Apps Script 편집기에 **붙여넣기 (Ctrl+V)**

> 💡 시트 ID는 제가 이미 코드 안에 박아넣었어요. 수정할 게 하나도 없어요!

**1-6.** 💾 저장 (Ctrl+S) — 프로젝트 이름 물어보면 "혜원이 영단어 서버" 같은 걸로 저장

**1-7.** 테스트:
- 상단 함수 드롭다운에서 **`testSheet`** 선택
- ▶️ **실행** 버튼 클릭
- "권한 필요" 창이 뜨면:
  - "권한 검토" 클릭 → 본인 구글 계정 선택
  - "확인되지 않은 앱" 경고 뜨면: **고급 → (안전하지 않은 페이지) 이동** → "허용"
- 스프레드시트 탭으로 돌아가서 **단어장기록 / 퀴즈기록** 시트가 생겼으면 ✅ 성공!

---

### 2단계: 웹앱으로 배포 (1분)

**2-1.** Apps Script 편집기 우측 상단 **"배포"** → **"새 배포"** 클릭

**2-2.** ⚙️ (톱니바퀴 아이콘) → **"웹 앱"** 선택

**2-3.** 설정:
- **설명**: 혜원이 영단어 시트 기록
- **실행 사용자**: 나
- **액세스 권한**: **모든 사용자** ⭐ (반드시!)

**2-4.** **"배포"** 클릭 → 또 권한 승인 (처음과 같은 방식)

**2-5.** **웹 앱 URL** 복사
- 형식: `https://script.google.com/macros/s/AKfyc**********/exec`

---

### 3단계: 퀴즈 HTML에 URL 붙여넣기 (30초)

**3-1.** `혜원이_영단어_월요일_퀴즈.html` 파일을 메모장 (또는 VS Code) 으로 열어요

**3-2.** **Ctrl+F**로 `GAS_URL` 검색 → 아래 줄 찾기:

```javascript
const GAS_URL = ""; // 예: "https://script.google.com/macros/s/AKfycbz.../exec"
```

**3-3.** 따옴표 안에 **2-5에서 복사한 URL** 붙여넣기:

```javascript
const GAS_URL = "https://script.google.com/macros/s/AKfycbxxxxx.../exec";
```

**3-4.** Ctrl+S 저장 ✅ **끝!**

> 💡 `LEGACY_APPS_URL`은 제가 이미 채워뒀어요 (재님이 기존에 쓰던 그 URL). 만지지 마세요!
