# 혜원이한테 링크만 보내기 📲 (구글 드라이브 방식)

## 🎯 목표
HTML 파일을 주고받는 게 아니라, **링크(URL)만 카톡으로 보내면** 혜원이가 클릭해서 바로 열리게 하기.

---

## 📝 최초 1회 설정 (10분)

### 1단계: 구글 드라이브에 폴더 만들기

**1-1.** https://drive.google.com 접속 → 재님 구글 계정 로그인

**1-2.** 왼쪽 상단 **"+ 새로 만들기"** → **"폴더"** 선택
- 폴더 이름: **`혜원이 영단어`**

**1-3.** 폴더 우클릭 → **공유** → **"일반 액세스" 를 "링크가 있는 모든 사용자"** 로 변경
- 권한: **"뷰어"** (혜원이는 보기만 가능, 수정은 못 하게)
- "완료" 클릭

> 💡 이렇게 한 번 설정하면 이 폴더 안에 들어가는 모든 파일이 자동으로 공유돼요!

---

### 2단계: 날짜별 서브 폴더 만들기

**2-1.** 방금 만든 `혜원이 영단어` 폴더 더블클릭해서 들어가기

**2-2.** **"+ 새로 만들기" → "폴더"** → 이름: **`20260421`**

**2-3.** `20260421` 폴더로 들어가기

---

### 3단계: 화요일 파일 업로드

**3-1.** 내 PC에서 `D:\Claude\05.혜원 영어\20260421\` 폴더 열기

**3-2.** 아래 **2개 파일**을 드래그해서 구글 드라이브 `20260421` 폴더에 떨어뜨리기:
- `혜원이_영단어_화요일_동물과자연2.html`  (아침 단어장)
- `혜원이_영단어_화요일_퀴즈.html`          (오후 퀴즈)

**3-3.** 업로드 완료될 때까지 기다리기 (퀴즈 파일이 228KB라 5초 정도)

---

### 4단계: 링크 복사 & 변형 ⚠️ **중요!**

구글 드라이브의 기본 링크는 "미리보기" 모드라 HTML이 **직접 실행되지 않아요**. 약간의 URL 마법이 필요해요!

**4-1.** 단어장 파일 우클릭 → **"링크 복사"** 클릭
- 복사되는 링크: `https://drive.google.com/file/d/XXXXXXXX/view?usp=sharing`

**4-2.** 이 링크에서 **파일 ID** 부분만 뽑아내요 (`/d/`와 `/view` 사이의 긴 문자열):
```
https://drive.google.com/file/d/[1AbcDefGhi2JklMno3PqrStu4VwxYz]/view?usp=sharing
                                  ↑ 이 부분이 파일 ID
```

**4-3.** **구글 드라이브 직접 뷰어 링크**로 변형:
```
https://drive.google.com/uc?export=view&id=파일ID
```

예시:
```
https://drive.google.com/uc?export=view&id=1AbcDefGhi2JklMno3PqrStu4VwxYz
```

> 🤔 **너무 복잡하다고요? 더 쉬운 방법 있어요!** → 아래 "초간단 방법" 참고

## 🎁 초간단 방법: 구글 드라이브 대신 **GitHub Pages** 쓰기

구글 드라이브는 HTML을 바로 실행하는 게 은근 까다로워요. 더 깔끔한 대안이 있어요.

### 방법 A: **GitHub Pages** (무료, 세팅 10분, 영구 사용)

**장점:**
- 한 번 설정하면 끝, 링크가 예쁘고 짧아요 (`https://이름.github.io/hyewon/20260421/`)
- 파일 올리면 자동으로 웹사이트가 됨
- HTML이 완벽하게 실행됨

**단점:**
- 최초 설정이 구글 드라이브보다 약간 복잡
- 깃허브 계정 필요 (이미 있으시면 GO!)

### 방법 B: **Netlify Drop** (무료, 세팅 1분, 가장 쉬움) ⭐⭐

1. https://app.netlify.com/drop 접속
2. `D:\Claude\05.혜원 영어\20260421\` 폴더를 화면에 **드래그해서 떨어뜨림**
3. 끝! 자동으로 링크 나옴 (예: `https://magical-starlight-abc123.netlify.app/혜원이_영단어_화요일_동물과자연2.html`)
4. 그 링크를 카톡으로 보내면 끝

**장점:** 정말 1분. 계정 없어도 가능 (단, 계정 만들면 영구 보관)
**단점:** 임시 URL은 24시간 뒤 사라질 수 있음 (계정 만들면 영구 OK)

### 방법 C: **재님 기존 "매일매일 퀴즈내기" 방식** ⭐⭐⭐

재님이 기존에 이미 쓰시던 방식이 있으시죠? 예를 들어:
- 카톡으로 HTML 파일 바로 전송
- 어딘가에 호스팅해둔 사이트
- 구글 드라이브로 이미 공유 중

**이미 쓰던 방식을 알려주시면 거기에 맞춰서 자동화까지 해드릴게요!**

## 📋 재님께 추천하는 순서

1. **오늘 화요일 파일 테스트**: 먼저 PC에서 `혜원이_영단어_화요일_동물과자연2.html`를 브라우저로 열어서 잘 작동하는지 확인
2. **퀴즈 테스트**: PC에서 `혜원이_영단어_화요일_퀴즈.html` 열어서 문제 풀어보기 → Google Sheets + 카톡 도착 확인
3. **전달 방식 결정**: 위 A/B/C 중 하나 선택 (Netlify Drop이 제일 쉬움!)
4. **혜원이 폰에 링크 전달**: 카톡 → "혜원이" 채팅방 → 링크 붙여넣기

## 💬 막힐 때

- "Netlify로 해봤는데 어려워" → 화면 캡처 보여주세요
- "GitHub 계정 만들었어" → 제가 Pages 설정 도와드림
- "그냥 카톡으로 파일 보낼래" → 그것도 OK! HTML 파일을 카톡 "나에게 보내기" 후 혜원이 폰으로 재전송

## 📱 혜원이 폰에서 잘 작동할까?

- **아이폰**: Safari에서 완벽 작동 ✅
- **갤럭시**: 크롬에서 완벽 작동 ✅
- **카톡 내장 브라우저**: 정상 작동하지만 음성(TTS)이 안 될 수 있음 → "브라우저로 열기" 추천
