4. **모든 코드 삭제** (Ctrl+A → Delete)
5. `D:\Claude\05.혜원 영어\20260427\GoogleAppsScript_코드.gs` 파일을 **메모장**으로 열어서 **전체 복사** (Ctrl+A → Ctrl+C)

### 4️⃣ ⏰ 매일 아침 7시 자동 실행 설정 (Time Trigger)

1. Apps Script 편집기 왼쪽 메뉴에서 **⏰ (트리거)** 아이콘 클릭
   - 없으면: 좌측 "알람시계" 아이콘
2. **➕ 트리거 추가** (우측 하단 파란 버튼)
3. 설정:

| 항목 | 값 |
|------|-----|
| 실행할 함수 | **`morningReminder`** |
| 실행할 배포 | `Head` (기본값) |
| 이벤트 소스 선택 | **시간 기반** |
| 시간 기반 트리거 유형 | **일일 타이머** |
| 시간 선택 | **오전 6시~7시** |
| 실패 알림 | 매일 (기본값) |

4. **저장** → 권한 한 번 더 승인

✅ 이제 **매일 아침 6~7시에 재님 카톡으로 자동 발송**!
